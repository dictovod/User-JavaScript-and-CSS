# User JS/CSS — UIAuto2

Скрипты для расширения [User JavaScript and CSS](https://chrome.google.com/webstore/detail/user-javascript-and-css/)
применяемые на сайте [uiauto2.devsleep.com](https://uiauto2.devsleep.com)

## Что делают скрипты

### uiauto2.devsleep.com
- Автоматически кликает на устройство в таблице
- Автоматически нажимает кнопку монитора (pi-desktop)
- Автоматически переходит на вкладку Shell
- Добавляет кнопку **Разбудить** (отправляет `input keyevent 26`)
- Добавляет поле **Обычный ввод** с кнопкой Send Input
- Сдвигает разделитель панелей на середину экрана

### uiauto.devsleep.com
- Блокирует `history.pushState` и `history.replaceState`

## Установка

1. Установите расширение [User JavaScript and CSS](https://chrome.google.com/webstore/detail/user-javascript-and-css/)
2. Откройте расширение → Import
3. Загрузите файл `user-js-css-v11-260612.json`
